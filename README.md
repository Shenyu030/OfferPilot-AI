# 🧠 OfferPilot-AI

AI求职助手（OfferPilot-AI）

基于 Streamlit + 通义千问（Qwen）开发的 AI 求职辅助工具，帮助求职者完成：

- 简历优化
- JD匹配分析
- 面试题生成
- PDF简历解析

---

## 项目背景

在求职过程中，经常需要针对不同岗位修改简历、分析岗位要求并准备面试。

因此开发了 OfferPilot-AI，希望利用大模型能力提高求职效率。

---

## 当前功能

### 1. AI简历优化

输入简历内容后：

- 提升专业表达
- 强化项目描述
- 优化成果展示
- 贴合AI相关岗位

---

### 2. JD分析

输入岗位JD：

- 提取核心能力要求
- 分析技能匹配度
- 给出补充建议

---

### 3. 面试题生成

根据岗位名称：

- 自动生成针对性面试问题
- 支持AI岗位
- 支持产品岗位

---

### 4. PDF简历解析

支持上传PDF简历：

- 自动提取文本
- 直接进入AI分析流程

---

## 技术栈

- Python
- Streamlit
- DashScope SDK
- Qwen-Turbo
- PyPDF2
